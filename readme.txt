Тестовое задание:
Разработать функционал стикеров (небольшие картинуки в уодном из углов товара).
Админ панель:
В "Каталог" должен появиться пункт "Стикеры". Работа со стикерами должна быть с схожа с работой остальной админки сайта. У пользователя не должно быть ощущение что их разрабатывал кто-то другой. При добавлениеи стикера есть два обязательных поля: "название" и "изображение", остальные поля на Ваш выбор, но если добавляете объяснить почему. Админка предпологает мультиязычность, этим можно принебречь.
На странице добавления/редактирования товара во владке "Данные" должен появитсься выбор стикера из ранее созданных.
Каталог, часть доступная для пользователей:
На странице категорий, на соответствующих товарах в правом верхнем углу изображения товара должно быть изобрание стикера высотой 50px.

База данных находится в папке dump_sql
